orvex.sh

Install endpoints for orvex AI tooling. Each script is a single bash file you can curl … | bash or download + inspect first.

BMAD modules (Linear + Docmost)

curl -fsSL orvex.sh/install/bmad | bash

Installs orvexai/bmad-modules into the current project, applies the Linear write-suppression patch, and stages override TOMLs. Requires Node ≥ 20.12 and SSH access to the orvexai GitHub org.
